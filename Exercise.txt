Mã hóa file
Sử dụng Base64 + Thay thế
Chuyển file về Base64 và lưu file
---------------
ĐỒ ÁN CÁ NHÂN GIỮA KỲ:
- Ứng dụng viết bằng java.
- 5 nội dung chính:
1. Các giải thuật cơ bản (dịch chuyển, thay thế,...). Chỉ thao tác với văn bản.
2. Thực hiện các giải thuật đối xứng (toàn bộ giải thuật java hỗ trợ) trong đó có 2 - 3 giải thuật tìm kiếm các thư viện hỗ trợ để làm. Cho phép chọn mode và padding. Thao tác với file và văn bản.
3. Thực hiện giải thuật mã hóa bất đối xứng. Chỉ cần thao tác với văn bản ngắn.
4. Thực hiện các giải thuật về hash. Tính toán giá trị hash của file hoặc văn bản.
5. Chữ ký điện tử: cơ chế phát sinh chữ ký và kiểm tra chữ ký.
Yêu cầu để được chấm bài giữa kỳ:
- Thiết kế giao diện swing.
- Mã hóa đối xứng và mã hóa bất đối xứng cho phép người dùng tạo key với chiều dài mà giải thuật hỗ trợ (Nếu người dùng có key sẵn thì nhập, không có thì tạo key). Cho phép lưu key về máy người dùng. (Key ở đây có thể là file, base64,...)
- Nộp file .jar hoặc .exe (nếu là exe thì được +1đ), 1 file hướng dẫn sử dụng, 1 file source code nén thành file .zip
